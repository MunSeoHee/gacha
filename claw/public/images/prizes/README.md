# 상품 이미지 에셋

결과 화면(공개된 카드)에 표시되는 상품 이미지를 여기에 넣습니다.
`src/config/prizes.ts` 의 `imageUrl` 경로와 파일명을 맞추면 자동으로 반영됩니다.

| 파일명 | 등급 |
|--------|------|
| `A.png` | A상 |
| `B.png` | B상 |
| `C.png` | C상 |
| `D.png` | D상 |
| `E.png` | E상 |
| `F.png` | F상 |

- 권장: 정사각형에 가까운 투명 배경 PNG (예: 512×512)
- 파일이 없으면 카드는 🎁 플레이스홀더로 표시됩니다.
- 상품을 등급별 여러 개로 늘리려면 `prizes.ts` 에 항목을 추가하고 각자 `imageUrl` 을 지정하세요.

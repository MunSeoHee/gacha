# 구체(Orb) 이미지 에셋

등급별 가운데 구체 이미지를 여기에 넣습니다. **파일명은 아래와 정확히 일치**해야 합니다(소문자).

| 등급 | 파일명 |
|------|--------|
| Normal | `normal.png` |
| Rare | `rare.png` |
| Epic | `epic.png` |
| Primeval | `primeval.png` |

경로: `public/images/orbs/<파일명>` → 앱에서 `/images/orbs/<파일명>` 로 로드됩니다.

## 권장 스펙
- **투명 배경 정사각형 PNG** (원형 구체 권장)
- 표시 크기 지름 160px 기준 → 선명하게 하려면 320px 이상(2x) 권장
- 파일이 없거나 로드 실패하면 **자동으로 기존 CSS 구체로 표시**됩니다 (에러 없음)

## 동작
- 이미지가 있으면 그 등급에서 이미지를 사용, 없으면 CSS 구체 유지
- 흔들림·부유·업그레이드(응축→펑)·버스트(깨짐)·주변 빛/파티클 연출은 이미지에도 그대로 적용
- 깨질 때는 화이트 플래시 + 파편 파티클로 처리(A 방식)되므로 별도 파편 이미지는 필요 없음
